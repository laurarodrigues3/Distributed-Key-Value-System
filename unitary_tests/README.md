# Diretório de Testes Unitários

Este diretório contém os testes unitários para validação do sistema.

## Conteúdo

- `api_tests.py` - Implementação dos testes da API REST
- `run_tests.py` - Script para execução dos testes
- `__init__.py` - Inicialização do módulo Python
- `README-tests.md` - Documentação detalhada dos testes

## Funcionalidades

- Testes de integração para os endpoints da API
- Validação dos health checks
- Testes de disponibilidade do sistema
- Relatório de resultados dos testes
- Execução automática durante o startup
